# 前置
```bash
基于[vue2] [vue cli 4.0](https://github.com/vuejs/vue-cli) 创建的项目框架。
```

# 技术栈
```bash
vue2.x
vuecli4.x
elementui2.x按需加载
echarts5按需加载
less
```

# 开发
```bash
# 建议yarn安装
# node15.x 版本及以上
yarn install | npm install  
yarn serve | npm run serve
```

# 打包
```bash
# 测试环境
yarn build:test

# 生产环境
yarn build
```
